# Keyboard Analytics Dashboard

A Linux desktop application for keyboard analytics and statistics visualization using Python, Tkinter, SQLite, and Matplotlib.

## Features
- Real-time keyboard statistics
- Interactive charts
- SQLite database logging
- Professional desktop dashboard
- Linux .deb package support

## Technologies
- Python
- Tkinter
- SQLite
- Matplotlib

## Installation

```bash
sudo dpkg -i keyinsight-dashboard.deb
